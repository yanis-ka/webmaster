Outils de placement en CSS

Les floatant

Le float est une propriété CSS qui permet de placer un éléments à droite ou à gauche de son conteneur

Flexbox

Le flex est une propriété CSS qui permet de placer les élément en ligne ou en colonne.

Il est très utilisé notamment pour créer des navbar ou mettre en ligne des éléments.
