Le CSS

Présentation et définition

CSS (cascading Style Sheets) est un langage de feuille de styles utilisé pour controller l'apparence visuelle des éléments HTML

Il existe 3 méthodes de déclarer du style

  1. Inline: Attribut style à l'intérieur de la balise
  2. Outside: dans une balise style pouvant être placée dans un body ou header
  3. External: dans un fichier externe "style.css" relié à ma page
